IRF projekt beadandó
Ötlet: Karácsonyi ajándékozást segítő program

Megvalósítás:
1/c XML beolvasás - Előzetesen felmért igények alapján a résztvevő diákok adatainak beolvasása
2/b Setter ág használata - Ajándék ötlet kereső: adott diákra rákeresve a program ötletet ad (előre megadott válaszok alapján), hogy milyen ajándékot szeretne az illető
3/a Írás CSV fájlba - A húzás eredményeit a program egy CSV fájlba írja ki az ellenőrizhetőség érdekében
4/c kódból generált UI - Keresősáv és résztvevők feltüntetése, valamint az ajándékötlethez szükséges grafikus elemek kidolgozása a főoldalon
